Architecture

System Architecture is a fairly standard web application model which is hosted on AWS services. The Web application server itself is hosted on an EC2 service, which calls an API known as Event Registry, an online API which scrapes news articles from hundreds of sources. For all the articles in a given story, each article is fed through the ML servers, hosted by Amazon Sagemaker for summarization, and all the articles together are sent in for clustering. Once all the articles are clustered, keywords are extracted from each cluster to help users identify the content of the clusters. The summaries, clusters, and tags are sent to the Amazon DynamoDB which talks to the web application servers and displays the information.
